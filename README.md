# LittleUselessBox
A tiny take on the useless box. Is an exersize in using the AVR sleep functions. 

##Parts list
* Arduino Pro Mini
* Self Resettable Switch (A8GS-1105)
* Connectors for Switch
* Pololu Step-up\Step-down Regulator(#2118)
* Adafruit Mini Lipo Charger ()
* Lipo Battery
